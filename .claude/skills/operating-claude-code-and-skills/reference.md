# Reference: Skill Information Architecture

## Principles
- **Progressive disclosure**：入口→必要に応じて詳細。
- **Single source**：定義/用語は `reference.md` に集約。
- **Consistency**：命名・見出し・用語を横並びのSkillと合わせる。

## Style
- 三人称・命令形は控えめ：説明は中立的に。
- 冗長な導入は不要。Claude は前提知識を持つ。
